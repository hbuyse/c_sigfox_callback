Compilation
===========

In order to compile the program, you simply have to do:

.. code:: bash

    make [OPTIONS]

Options available :
    OPTIM=NONE|DEBUG|SIZE|SPEED   (dft : DEBUG)
    STATIC=0|1                    (dft : 0)
    V=0|1                         (dft : 0)


Utilization
===========

To use the program:

.. code:: bash

    ./sigfox_callback.out
